# reports/

Evaluation outputs and deliverables. JSON and CSV files are generated by the pipeline. The DOCX, and PPTX are the final submission deliverables.

## Evaluation results (JSON)

| File | Description |
|---|---|
| `ragas_v1.json` | RAGAS scores for pipeline v1 (n=83 questions) |
| `ragas_v2.json` | RAGAS scores for pipeline v2 (n=83 questions) |
| `ragas_v3.json` | RAGAS scores for pipeline v3 (n=84 questions) |
| `ragas_v1_*.csv` | Per-question RAGAS detail for v1 |
| `ragas_v2_*.csv` | Per-question RAGAS detail for v2 |
| `ragas_v3_*.csv` | Per-question RAGAS detail for v3 |
| `tool_call_acc_v1.json` | Tool call accuracy for v1 |
| `tool_call_acc_v2.json` | Tool call accuracy for v2 - overall 67.9% |
| `tool_call_acc_v3.json` | Tool call accuracy for v3 - overall 70.2% |
| `haic_v1.json` | HAIC evaluation v1 (4-metric format) |
| `haic_v2.json` | HAIC evaluation v2 (judge score + accept rate) |
| `haic_v3.json` | HAIC evaluation v3 (n=84, mean judge score 3.79/5) |

## Deliverables

| File | Description |
|---|---|
| `PaperPilot_Report.docx` | Final report document |
